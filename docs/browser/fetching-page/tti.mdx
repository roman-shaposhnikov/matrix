# TTI (Time to Interactive)

[Time to Interactive](https://web.dev/interactive/) — проверяет, за какое время страница станет интерактивной.

Страница считается интерактивной если:<br/>
The page displays useful content, which is measured by the First Contentful Paint<br/>
Event handlers are registered for most visible page elements<br/>
The page responds to user interactions within 50 milliseconds
